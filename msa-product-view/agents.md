
TypeScript: Главный язык разработки.

React + Next.js / Remix: Самая востребованная связка на рынке. React активно использует серверные компоненты (React Server Components), перенося часть вычислений на бэкенд.

Инфраструктура и инструменты сборки
Vite: Стал основным и обязательным сборщиком

🛠️ Выбор стека технологийФреймворк: React (через Vite для максимальной скорости сборки).
Библиотека графиков: recharts — построена на SVG, имеет отличную производительность, легко кастомизируется и плавно анимирует изменения «из коробки».
Стилизация: Tailwind CSS — для быстрой адаптивной сетки (grid) 3х3.
